Welcome to the relation normalzation tool!

This tool is designed mainly to help students learn functional dependencies, normal forms, and normalization. It can also be used to test your table for normal forms or normalize your table to 2NF, 3NF or BCNF using a given set of functional dependencies.

Anyone is welcome to use the tool! The first time you use it, please refresh your browswer after loading the system.
